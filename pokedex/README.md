# Real Life Pokédex 🔴

Point it at any photo — animal, plant, object, food, landmark — and get a full Pokédex entry generated by Claude's vision API.

## Setup

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY="your-key-here"
```

## Usage

```bash
# Local file
python pokedex.py photo.jpg

# Public image URL
python pokedex.py https://example.com/lion.jpg

# Disable streaming (wait for full response)
python pokedex.py photo.png --no-stream
```

## How it works

`pokedex.py` sends your image to **Claude Opus 4.6** with a carefully crafted system prompt that instructs it to act as a Pokédex. Claude analyses the image using its vision capabilities and returns a structured entry with:

- A creative Pokémon species name
- Thematically appropriate types
- Estimated height and weight
- Flavour text in classic Pokédex style
- Three abilities matching the subject's real traits
- Base stats reflecting real-world characteristics
